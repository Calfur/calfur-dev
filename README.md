## How to preper a new server

### Install K3s

curl -sfL https://get.k3s.io | INSTALL_K3S_EXEC="server --disable traefik" sh -s -

### How to uninstall K3s
https://docs.k3s.io/installation/uninstall

/usr/local/bin/k3s-uninstall.sh
